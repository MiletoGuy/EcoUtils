# v0.2.9

Suporte inicial ao Firebird 5.0. É possível configurar a versão do Firebird por instância e definir portas e DLLs para cada versão nas Configurações.

---

## FEATURES

- Ao criar ou editar uma instância, novo seletor **Firebird 2.5 / Firebird 5.0** define qual versão será usada por aquela instância
- O painel de Configurações ganhou a seção **Firebird** — permite configurar a porta e o caminho da `fbclient.dll` para cada versão (2.5 e 5.0)
- Ao salvar as Configurações com uma porta alterada, todos os `.ini` das instâncias afetadas são atualizados automaticamente
- O `.ini` de cada instância passa a incluir a seção `[Eco]` com as configurações de versão, porta e servidor do Firebird
- `fbclient.dll` encontrada na pasta legada (`C:\ecosis\windows\`) é migrada automaticamente para a pasta estruturada ao confirmar uma instância com Firebird 2.5
