# v0.2.2

Agora é possível configurar parâmetros do `.ini` de cada instância diretamente no flyout, sem editar o arquivo manualmente.

---

## FEATURES

- Seção **Configurações do .ini** colapsável no flyout de instância, oculta por padrão para não poluir o formulário
- Parâmetros editáveis da seção `[PREFERENCIAS]`: Usuário (texto), PesquisaTotalDosProdutos, MonitorarTempoSelects, SincronizaTabelaPreco e MultiplasInstancias (checkboxes S/N)
- Ao criar uma instância, os parâmetros são gravados no `.ini` gerado; ao editar, o `.ini` existente é atualizado preservando todas as demais seções intactas
- Ao editar uma instância existente, os valores atuais do `.ini` são carregados automaticamente no formulário
