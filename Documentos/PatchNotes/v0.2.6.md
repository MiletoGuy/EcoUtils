# v0.2.6

Nova aba **Banco de Dados** na sidebar para executar queries diretamente nos bancos ECO. O painel de Configurações também ganhou uma aba de Patchnotes e agora fecha ao clicar fora.

---

## FEATURES

- Nova aba **Banco de Dados** na sidebar — selecione um banco `.eco`, execute queries SQL e visualize os resultados sem sair do EcoUtils
- **Executor SQL livre** — editor de texto com execução por `Ctrl+Enter` e visualizador de resultados com ordenação por coluna e filtro em tempo real
- **Gerenciamento de transação** — comandos de escrita (`INSERT`, `UPDATE`, `DELETE`, etc.) abrem uma transação explícita; botões **Commit** e **Rollback** ficam habilitados na toolbar enquanto há alterações pendentes
- **Exportação de resultados** — copiar para clipboard (TSV) ou exportar arquivo `.csv`
- Indicador de status de conexão ao selecionar o banco (conectado / erro / conectando)
- Nova aba **Patchnotes** no painel de Configurações — exibe o histórico de versões com resumo, features e bugs corrigidos de cada release
- Clicar fora do painel de Configurações agora o fecha automaticamente
