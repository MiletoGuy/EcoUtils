# v0.2.0

Agora é possível importar bancos de dados e executáveis ECO diretamente pelo Utils, sem precisar copiar arquivos manualmente.

---

## FEATURES

- Importação de banco de dados pelo botão **+** no flyout de instância: aceita `.eco`, `.fbk`/`.gbk` ou pacote compactado (`.zip`, `.rar`, `.7z`)
- Restauração automática de backup Firebird com barra de progresso em tempo real
- Importação de executável ECO pelo botão **+** no flyout: aceita `eco.exe` avulso ou dentro de pacote compactado
- Aviso de confirmação ao importar um executável que já existe, com opção de substituir

---

## BUGS CORRIGIDOS

- Crash ao abrir o flyout de nova instância em algumas situações
