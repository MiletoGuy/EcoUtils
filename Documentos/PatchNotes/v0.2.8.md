# v0.2.8

Novo recurso para forçar a versão do banco de dados para fins de teste, com suporte a restaurar o valor original quando necessário.

---

## FEATURES

- Ao editar ou criar uma instância, novo checkbox **"Usar versão do executável"** permite forçar o campo `VERSAO` da `TGERLICENCA` para corresponder à versão do executável selecionado
- Instâncias com a versão do banco alterada exibem um indicador **✎** azul na coluna de status da tabela
- Nova coluna **Versão original** na tabela de instâncias (oculta por padrão, habilitável pelas configurações de colunas) — exibe o valor original do campo antes da alteração
- No flyout de edição, botão **"Restaurar versão original"** reverte o banco para o valor registrado na adição da instância
