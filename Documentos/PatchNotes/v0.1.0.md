# v0.1.0

Versão inicial do EcoUtils. Permite cadastrar, editar e excluir instâncias ECO e iniciá-las diretamente pelo app, além de suportar atualização automática quando houver nova versão disponível.

---

## FEATURES

- Cadastro, edição e exclusão de instâncias ECO com apelido, executável e banco de dados
- Execução de instâncias diretamente pelo app
- Exibição da versão do banco Firebird na listagem
- Barra de pesquisa para filtrar instâncias por apelido, executável ou banco
- Visibilidade configurável de colunas na listagem
- Atualização automática do app quando uma nova versão for lançada
